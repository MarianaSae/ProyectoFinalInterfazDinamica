# Dino Runner Canvas (versión Banana Runner)

Proyecto final de videojuego usando **HTML, CSS, JavaScript y Canvas** con una estética clara y un personaje tipo bolita que esquiva bananas.

## Cambios visuales principales

- Tema completamente en **colores claros** (fondos blancos y azules suaves).
- El personaje ahora es una **bolita con patas** en lugar de un cuadrado.
- Los obstáculos que hay que saltar ahora son **bananas** dibujadas en Canvas.
- El área del juego (canvas) es más grande: `960x360`.
- El perfil del usuario se organiza en un **grid de tres bloques** (datos, estadísticas, acciones).

El resto de la lógica (usuarios, perfil, partidas, historial) funciona igual que en la versión anterior.

## Cómo usar

1. Abre `index.html` en tu navegador.
2. Regístrate con nombre de usuario, correo, contraseña y color favorito.
3. Inicia una partida desde el perfil.
4. Presiona **Espacio**, **Flecha arriba** o toca el Canvas para saltar.
5. Evita las bananas y llega al puntaje más alto posible.

Puedes subir este mismo proyecto a tu repositorio de GitHub como entrega del proyecto final.
